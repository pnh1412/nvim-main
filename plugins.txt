Recent config/keymap notes
- <leader>la is reserved for Lspsaga code action UI.
- <leader>lA calls native vim.lsp.buf.code_action.
- blink.cmp manual completion fallback keys: <C-Space>, <M-;>, <M-c>.
- Filetype-local UI/options were moved from lua/core/autocmds.lua into ftplugin/.
- alpha-nvim loads on VimEnter instead of startup.
- tree-sitter-norg and tree-sitter-norg-meta lazy-load with Neorg/norg files.
- nvim-ts-autotag lazy-loads only for HTML/JSX/TSX/Vue/Svelte/XML style buffers.
- package-info.nvim lazy-loads for package.json workflows under <leader>np.
